When I first learnt about margin, border, and padding it was on freecodecamp while working through the Basic CSS module. Even after building a Caf√© Menu and a Set of Coloured Markers I still wasn't sure which was which. Then I moved onto learning on DevAcademy Foundations and it clicked. It is similar to creating a word document where the Margin is the outside space around the page, the border is a border you can put around the outside of your writing/content, while the padding is the space between the border and the writing/content. I don't know why this was so difficult to understand during freecodecamp. Likely because it was easier to just read the hints. Anyway let's go through it.

On a word document the margin can be set by sliding the dark space on the ruler at the top and to the left of the document screen. A border is set on the inside of the inside of the margin when a border is added to which you can format its thickness, colour and style. And padding is created by dragging the arrows along the ruler at the top and left of the document screen.

When coding these we can create the content as an html and change it's margin, border and padding through through CSS.
